## Installation Instructions
To install Nexdeen AI, follow these steps:
1. **Clone the repository**:
   ```
   git clone https://github.com/maninizinlhwari-source/nexdeen-ai.git
   ```
2. **Navigate to the directory**:
   ```
   cd nexdeen-ai
   ```
3. **Install dependencies**:
   ```
   npm install
   ```

## Usage Guide
To start using Nexdeen AI, follow these instructions:
- Run the application using:
  ```
  npm start
  ```
- Open your web browser and go to `http://localhost:3000` to access the application.

## Customization Options
Nexdeen AI can be customized through the following options:

- **Themes**: Change themes by modifying the `theme` property in `config.js`.
